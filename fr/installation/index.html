Installer Onyx sur un Raspberry Pi

Pour installer Onyx sur votre Raspberry Pi vous pouvez tout simplement suivre cette vidéo :


Cependant si vous souhaitez un tutoriel écrit le voici :

Tout d'abord veuilles vous rendez sur la page de téléchargement : https://onyxlabs.fr/fr/download

Téléchargez Onyx et Onyx Finder et installez les. Ensuite branchez votre carte sd dans votre ordinateur et formattez la si besoin.


Lancez onyx-installer et suivez les différentes étapes pour mettre Onyx sur votre carte SD.

Ensuite, branchez le raspberry pi en Ethernet ou en wifi en suivant les indications dans une prochaine vidéo, et branchez le sur le secteur. Il vous suffit maintenant d'attendre que le Raspberry Pi redémarre pour mettre à jours vos paramètres, vous pouvez d'ailleurs suivre l'avancement en branchant votre Pi à un écran.

Accéder à Onyx

Maintenant, pour vous connecter au Raspberry Pi deux choix s'offre à vous : chercher l'ip grâce à votre box ou à différents logiciels ou utiliser Onyx Finder, je vais choisir la seconde option.

Ouvez donc Onyx finder vous allez tomber sur cette fenetre :

Cliquez simplement sur Find Onyx et attendez qu'il le trouve, ensuite vous verrez cela :

Vous pouvez donc maintenant vous connecter à Onyx en cliquant sur Open Onyx, il vous suffit enfin de suivre les différentes étapes sur votre navigateur pour finir la configuration de base d'Onyx !
