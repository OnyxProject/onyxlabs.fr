Installer une compétence sur Onyx

Pour installer une compétence, rendez-vous tout d'abord sur l'onglet compétence d'Onyx :


Puis dans l'onglet Installer et séléctionnez la compétence que vous voulez installer et cliquez simplement sur Installer, c'est tout simple.


Cependant quelques compétences auront besoin d'une configuration supplémentaire de votre part, pour savoir laquelle veuillez vous rendre sur le github de la compétence, pour voir les différentes configurations si il y en a, voici d'ailleurs le lien où il y a le github de toutes les compétences validées par notre équipe : Github.
